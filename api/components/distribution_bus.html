Distribution Bus#

pydantic model gdm.DistributionBus#

Interface for distribution bus.

Examples

>>> from gdm import DistributionBus
>>> DistributionBus.example()
field coordinate: Location | None = None#

Coordinate for the power system bus.

field feeder: Annotated[Optional[DistributionFeeder], Field(None, description='Name of the feeder.')] = None#

Name of the feeder.

field nominal_voltage: PositiveVoltage [Required]#

Nominal voltage for this bus.

field phases: list[Phase] [Required]#

List of phases for this bus.

field substation: Annotated[Optional[DistributionSubstation], Field(None, description='Name of the substation.')] = None#

Name of the substation.

field voltage_type: VoltageTypes [Required]#

Voltage types for buses.

field voltagelimits: list[VoltageLimitSet] = []#

List of voltage limit sets for this bus.
