Load Equipment#

pydantic model gdm.PhaseLoadEquipment#

Interface for single phase load equipment. Uses ZIP model where real power is: P = P_0[ a_p (|V|/V_0)^2 + b_p (|V|/V_0) + c_p] and reactive power is: Q = Q_0[ a_q (|V|/V_0)^2 + b_q (|V|/V_0) + c_q]
